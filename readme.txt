🧘‍♀️ Kula Annex – Yoga & Meditation Website
Welcome to Kula Annex, a beautifully designed, responsive yoga and meditation studio website built using HTML and CSS. This site showcases a yoga center’s offerings, classes, pricing, instructors, and member perks with a clean, modern UI inspired by mindful living and well-being.

🚀 Features
🧘 Clean and professional layout for yoga studios

📱 Fully responsive design

🗓️ Class types and schedules

💰 Pricing plans & intro offers

🎥 Integrated "Watch Video" button

🌍 Online & in-studio options

❤️ Member perks section

🖼️ Elegant use of SVGs and quality images


🛠️ Built With
HTML5

CSS3

Google Fonts (Lato, Montserrat)

Flexbox & Grid for layout

🔧 Getting Started

To run this project locally:

1. Clone the repo:
   git clone https://github.com/Aryan-The-Ghost/Kula-Annex.git

2. Navigate to the project folder:
   cd Kula-Annex
   
3. Open index.html in your browser:
   open index.html  # MacOS
   start index.html # Windows
